# Literature Notes

One file per paper (or per tight cluster of 2–3 related papers).

## Convention

- Filename: `NN_short-slug.md` (NN = reading order, 01, 02, …)
- Each file:
  - Full citation (BibTeX at the bottom helps for later thesis writing)
  - 1-paragraph problem framing
  - Method summary (bullets)
  - Metrics & dataset
  - What's relevant to Meditab (1–3 bullets)
  - What's NOT relevant (helps scoping)
