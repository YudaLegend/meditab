# Meditab

Clinical information extraction from Catalan free-text patient histories.

TFM (Master's Thesis) project. Private repository — contains no patient data in version control. See `HANDOFF.md` for current status and `docs/scope.md` for project scope.

## Structure

```
meditab/
├── HANDOFF.md              # session-to-session status — read first
├── docs/                   # scope, annotation schema, eval plan, governance
├── thesis/                 # thesis document & literature notes
├── src/                    # source code
├── notebooks/              # exploratory notebooks (no data checked in)
├── eval/                   # evaluation harness + results
└── data/                   # GITIGNORED — local patient data only
```
